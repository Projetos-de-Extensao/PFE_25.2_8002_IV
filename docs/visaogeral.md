# Visão Geral do Projeto: Gestão de Monitores

# 1. O Desafio

Este projeto foi desenvolvido como parte da disciplina de **Projetos de Extensão (PFE)**. O objetivo principal é atender a uma demanda real para modernizar o processo de **Gestão de Monitoria**.

Atualmente, o controle de monitores, disciplinas e alocações de horários muitas vezes é feito de forma manual ou descentralizada. O desafio proposto foi criar uma interface web amigável que centralize essas informações.

# 2. A Solução Proposta

Desenvolvemos uma aplicação **Front-end em React** que serve como painel administrativo para a coordenação e gestão. A solução foca na experiência do usuário (UX/UI), garantindo que tarefas complexas sejam feitas de forma intuitiva.

# Principais Funcionalidades:
* **Dashboard:** Visão rápida do status das monitorias.
* **Gestão de Monitores:** Cadastro e visualização de alunos monitores.
* **Alocação:** Ferramenta para vincular monitores às disciplinas ofertadas.
