# Ambiente de Desenvolvimento

Para rodar ou editar este projeto, você precisará ter as seguintes ferramentas instaladas no seu computador.

# Pré-requisitos

* **Node.js** (versão 16 ou superior): Essencial para rodar o React.
* **NPM** (Node Package Manager): Geralmente já vem instalado junto com o Node.js.
* **Git**: Para baixar o código e gerenciar versões.
* **Visual Studio Code (VS Code)**: O editor de código recomendado.

# Extensões Recomendadas (VS Code)

Para facilitar o desenvolvimento, sugerimos as extensões:
* **ESLint**: Para encontrar erros no código JavaScript/React.
* **Prettier**: Para formatar o código automaticamente.
